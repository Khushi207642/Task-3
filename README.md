# Penetration Testing Toolkit

Project Name: Penetration Testing Toolkit

## Description
This is a Python-based modular penetration testing toolkit created for educational purposes only.

## Modules
1. Port Scanner – scans open ports on a given host
2. Brute Force Demo – demonstrates how password attempts work using a dummy password list

## How to Run
1. Open terminal or command prompt
2. Navigate to the Task-3 folder
3. Run:
   python main.py
